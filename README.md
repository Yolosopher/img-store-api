# img-store API

img-store-api is a simple image store API built with Node.js and Express, designed to allow users to upload, manage, and access images via a RESTful API.

## Features

- User authentication and authorization.
- Image upload as private or public.
- API tokens generation for authentication.
- Capability to manage uploaded images.

## Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd img-store
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Build the project:

   ```bash
   npm run build
   ```

## Testing

```plaintext
Run tests using:
```
